## Installation & Usage

To install Begynner, run the following command:

```bash
npm i -g begynner
```

This will create the beg command that can be used in your [terminal emulator](https://en.wikipedia.org/wiki/Terminal_emulator).

To start a new project, simply type beg in the terminal emulator and provide your GitHub username and a directory name, or use the suggested name. Select the project template (index) and your new project is ready to go! ðŸŽ‰
